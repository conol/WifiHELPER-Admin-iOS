## Installation

CORONAWriter is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "CORONAWriter", :git => 'https://gitlab.conol.jp/IoT/CORONAWriter.git', :branch => 'develop'
```
