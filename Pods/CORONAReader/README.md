## Installation

CORONAReader is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "CORONAReader", :git => 'https://gitlab.conol.jp/IoT/CORONAReader.git', :branch => 'develop'
```
